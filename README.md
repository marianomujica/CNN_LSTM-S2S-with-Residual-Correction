# CNN_LSTM-S2S-with-Residual-Correction
Demand forecasting pipeline.
Files:
1) CNN_LSTM S2S_Corr: main code for dataset preparation and model training.
2) Data_Ibarlucea: .csv files with 15 minute resolution of Power and Energy data provided by Cooperativa Electrica y de Consumo from Ibarlucea, Santa Fe,
Argentina.

Meteorological Data can be downloaded from National Meteorolgical System (SMN) webapage with the main code by setting up the desired date range.

Plots are automatically generated as html files by running the main code.
Trained models are saved as .joblib files by running the main code.
